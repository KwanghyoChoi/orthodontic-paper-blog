## 저자 정보 규칙

**서두**: 가볍게 자기소개로 시작
```markdown

```


## 스타일 가이드

### 제목 작성법

**좋은 예**:
- "투명교정 정확도, 실제로 얼마나 될까? - 2024 연구 분석"
- "소아 인비절라인에서 IPR 타이밍이 중요한 이유"
- "치아 이동 예측 vs 실제, 그 차이를 줄이는 방법"

**피해야 할 예**:
- "Clear Aligner Treatment Accuracy에 대한 고찰" (너무 학술적)
- "꼭 알아야 할 교정 정보!" (너무 가벼움)

### 문단 작성

- 한 문단 3-4문장 이내
- 첫 문장에 핵심
- 모바일 가독성 고려

### 데이터 표현

```markdown
❌ "p<0.05로 유의했다"
✅ "치료 기간이 평균 3개월 단축되었으며, 이는 통계적으로 의미 있는 차이였다"

❌ "높은 정확도를 보였다"  
✅ "78%의 정확도를 보였는데, 이는 기존 연구(65%)보다 13% 높은 수치다"
```

### 전문용어 처리

첫 등장 시 풀어쓰기:
```markdown
인접면 삭제(IPR, Interproximal Reduction)를 시행한 결과...

이후부터는 "IPR"만 사용
```

### 이미지 캡션

```markdown
![치료 전후 비교](images/fig_1.png)
*그림 1. 투명교정 치료 전(좌)과 후(우) 비교. 상악 전치부 배열이 개선됨. 
(Source: Kim et al., Journal of Orthodontics, 2024)*
```

## 플랫폼별 추가 설정

### 네이버 블로그

- 이미지: 가로 최대 860px
- 키워드: 검색 노출을 위해 제목에 핵심 키워드 포함
- 해시태그: 본문 하단에 #인비절라인 #투명교정 #교정연구 형식

### 워드프레스

- 이미지: 가로 최대 1200px, webp 형식 권장
- SEO: Yoast 플러그인 호환 메타데이터
- 카테고리/태그 시스템 활용

---

## 분량 규칙

- **최소 3,000자 이상** (전문성 있는 논문 기반 블로그)
- 최신 연구 비교 시 각 연구별 상세 데이터 포함
- 표/테이블 활용하여 데이터 가독성 확보

## 체크리스트

발행 전 확인:

- [ ] **서두에 저자 자기소개가 있는가** (가볍게)
- [ ] **마지막에 저자 정보가 정확히 있는가** (이름, 전문의, 병원, 지역)
- [ ] **원논문 정보가 정확히 명시되어 있는가** (저자, 제목, 저널, DOI)
- [ ] **3,000자 이상인가**
- [ ] 제목이 검색 친화적인가
- [ ] 핵심 요약이 3줄 이내인가
- [ ] 모든 전문용어에 첫 등장 시 설명이 있는가
- [ ] 데이터가 구체적인가 (숫자, %)
- [ ] 이미지에 인용 표기가 있는가
- [ ] 임상 적용 가이드가 명확한가
- [ ] 한계점이 언급되었는가
- [ ] 참고문헌 형식이 일관적인가
- [ ] 맞춤법 검사 완료
- [ ] 모바일에서 미리보기 확인
